# CLI Shell Interpreter (C++)

Простой интерпретатор командной строки с поддержкой пайплайнов,
переменных окружения и встроенных команд.

## Команда

- Артамонова Анна
- Бурмистрова Юлия
- Малышева Анастасия

## Поддерживаемые команды

- `cat` — вывод содержимого файла
- `echo` — вывод аргументов
- `wc` — строки/слова/байты
- `pwd` — текущая директория
- `exit` — завершение shell

## Возможности

- одинарные и двойные кавычки
- подстановка `$VAR`
- присваивание переменных `NAME=value`
- пайплайны `|`
- запуск внешних программ
- переменные окружения

## Архитектура

Проект реализуется как модульный интерпретатор командной строки.  
Архитектура разделена на независимые компоненты с чёткой ответственностью.

Основные подсистемы:

- **Shell / REPL** — точка входа, цикл чтения команд, хранение окружения и кодов возврата.
- **Frontend** — фасад для анализа ввода.
  - **Lexer** — разбор строки на токены с учётом кавычек и `$`-подстановок.
  - **Parser** — построение AST (Pipeline / CommandSpec).
- **Executor** — выполнение команд и пайплайнов, работа с `fork/exec/pipe/dup2/wait`.
- **BuiltinRegistry** — встроенные команды (`echo`, `cat`, `wc`, `pwd`, `exit`).
- **ExternalRunner** — запуск внешних программ через `execve` и поиск в `PATH`.
- **Environment** — хранение и модификация переменных окружения.

Архитектура проектируется заранее и может уточняться в процессе реализации,
однако основные интерфейсы и ответственность компонентов фиксированы.
